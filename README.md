## 如何使用：

克隆本仓库

下载gpuz软件，将exe文件放至本项目根目录

运行start.bat

打开VRChat，确保OSC已打开。硬件信息会自动发送至聊天框

## How to use:

Clone this repository

Download the gpuz software and place the .exe file in the root directory of this project

Run start.bat

Open VRChat and ensure OSC is enabled. Hardware information will be automatically sent to the chat window.
